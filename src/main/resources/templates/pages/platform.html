Principle Software Engineer - Spring Framework

San Francisco

The Spring engineering team is looking for an exceptional and passionate developer to contribute to the Spring Framework with a primary focus on web development. In this role, you will be a core member of the Spring JS team, which is responsible for ensuring Spring is the best platform for server-side applications no matter the choice of client-side technologies. We develop JavaScript libraries within the cujoJS project. We believe deeply in the Open Web, utilizing open technologies, contributing to open source projects and participating in creating open standards.

Requirements:

  • BS/Masters in Computer Science or equivalent work experience
  • Strong background in web application development.
  • 8+ years of software development experience.
  • Excellent programming skills in Java and JavaScript.
  • Experience with front end technologies such as HTML and CSS
  • Experience working with and contributing to open-source software projects.
  • Strong communication skills and ability to collaborate remotely.
  • Mentoring/coaching skills
  • Presentation skills

Desired Skills:

  • Experience with the Spring Framework, including Spring MVC
  • Familiarity with Inversion of Control, Dependency Injection, Aspect Oriented Programming and Test Driven Development
  • Ability to reason about highly asynchronous code
  • Familiarity with cujoJS
  • Familiarity with other client side libraries, including: Dojo, jQuery, Sencha/Ext, Backbone.js or AngularJS, is a plus
