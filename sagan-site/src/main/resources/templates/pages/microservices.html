Microservices

Microservice architectures are growing in popularity. Spring can help you write microservices that are small and self-contained and has many tools to help you look after your microservices once they’re in the hands of your users.In addition, Spring Cloud can help your microservices work together so you can reliably deliver rich functionality to your users.

What are microservices?

Microservices are a modern approach to software whereby application code is delivered in small, manageable pieces, independent of others.

Why build microservices?

Their small scale and relative isolation can lead to many additional benefits, such as easier maintenance, improved productivity, greater fault tolerance, better business alignment, and more.

Microservices with Spring Boot

With Spring Boot, your microservices can start small and iterate fast. That’s why it has become the de facto standard for Java™ microservices. Quickstart your project with Spring Initializr and then package as a JAR. With Spring Boot’s embedded server model, you’re ready to go in minutes.

Try our quickstart guide

Microservice resilience with Spring Cloud

The distributed nature of microservices can bring challenges. Spring can help you mitigate these issues. With several ready-to-run cloud patterns, Spring Cloud can help with service discovery, load-balancing, circuit-breaking, distributed tracing, monitoring, and can even act as an API gateway.

Build streaming data microservices with Spring Cloud Stream

Spring Cloud Stream makes it easy to consume and produce events no matter which messaging platform you choose. Designed to help build highly scalable, event-driven systems, Spring Cloud Stream can connect your microservices with real-time messaging in just a few lines of code.

Get started with Spring Cloud Stream

Manage your microservices

Micrometer, Spring Boot’s optional instrumentation framework, sends valuable metric insights straight to Prometheus, Atlas, and more. Spring Cloud’s Sleuth and Zipkin projects offer distributed tracing so that you can follow along with what’s happening in real time.

Get started with Micrometer on Spring Boot

Microservices on Cloud Foundry

Because microservices tend to be small and stateless, they can scale horizontally. Platforms like PAS and PKS can do this with ease and greatly reduce your administrative overhead. With cloud connectors, you can even consume multiple backend services effortlessly.
