If you want to install STS 3 in an existing Eclipse installation, you can use one of the following update sites. Please choose the one that matches the Eclipse version you're running:

Eclipse Update Sites
4.9 http://download.springsource.com/release/TOOLS/update/e4.9/

As an alternative, you can also install the STS 3 components as add-ons into an existing Spring Tools 4 installation via the Eclipse Marketplace.
